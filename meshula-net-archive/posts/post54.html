X3D

2007-04-15 14:15:18

This article offers no commentary, it is simply part of my continuing theme of convergence technologies. This article is a small collection of links to do with another 3D content distribution technology: X3D.

X3D is an open standards deployment technology for interactive 3D technology. It picks up where VRML left off, moving VRML's nodes to XML schema technology. An overview is available here. Mashups of differing web content, such as Google Maps, and 3d models, are a compelling application enabled by X3D, but not by most other playback engines.

The X3D specification is hosted here.

Flux player - web browser X3D playback engine. Flux Studio is a modeling and animation package targeting X3D. Engine is opensource (LGPL) hosted on sourceforge. Written in C#, lively contributor community. X3D was ratified as an ISO standard in 2004.

SwirlX3D - editor from Pinecoast Software targeting VRML & X3D

WireFusion - proprietary player incorporating many playback technologies, including X3D

Xi3D - Java based rendering library for X3D, doesn't incorporate more recent X3D nodes.

Some consulting companies are offering interesting X3D services, for example Octaga.

The web3d consortium hosts X3D-Edit.

PaperVision3D is an open source initiative to integrate Flash with X3D.
