The source code for a 68k forth interpreter/compiler called Fusion Forth that Don Palmer and I wrote. It has some Atari ST specific extensions in it, but it is really easy to port to any 68k based embedded system. Also included are an IDE, some graphic demos, a font editor, and a text editor, all written in Forth. Despite being created for the Atari ST, the code is still a good read if you've ever been curious about how a threaded interpreter is built from the ground up.
